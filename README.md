# n8n Automations
Eine wachsende Sammlung praxisorientierter Workflows gebaut mit [n8n](https://n8n.io) – einem Open-Source-Tool für Prozessautomatisierung.
Der Fokus liegt auf Automatisierungen die im Alltag echten Mehrwert schaffen: weniger manuelle Arbeit, schnellere Reaktionszeiten und intelligentere Prozesse.

## Workflows

| Workflow | Beschreibung | Tools | Status |
|----------|-------------|-------|--------|
| [Gmail Alert](./gmail-alert/) | Echtzeit-Benachrichtigung bei dringenden E-Mails bestimmter Absender | Gmail API, Telegram | ✅ Fertig |
| [Brand Monitoring](./brand-monitoring/) | Automatisches Marken-Monitoring mit KI-Sentiment-Analyse, Live-Dashboard und wöchentlichem Report | Claude AI, Google Alerts, Google Sheets, Looker Studio, Telegram, Gmail | ✅ Fertig |
| [Produkttextgenerator](./product-text-generator/) | Automatische Produkttextgenerierung per Formular – direkt in WooCommerce als Entwurf | Claude AI, WooCommerce, Google Sheets, Yoast SEO | ✅ Fertig |
| [Rechnungsverarbeitung](./invoice-processing/) | Automatische Extraktion von Beträgen und Produktarten aus Outlook-Rechnungsmails per KI – Ergebnis als strukturierte Datei in Google Drive | Claude AI, Outlook, Google Drive | ✅ Fertig |
| [Content Pipeline](./content-pipeline/) | End-to-End Content-Automatisierung: von SEO-Analyse über Themengenerierung bis zur automatischen Aufbereitung für Blog, Instagram und Pinterest | Claude AI, Google Search Console, Google Analytics, Google Keyword Planner, Google Trends, WordPress, Google Sheets | 🚧 In Arbeit |

## Technologie
Alle Workflows laufen auf n8n – lokal installiert, datenschutzfreundlich, keine Cloud-Abhängigkeit.
KI-gestützte Schritte verwenden die Claude API (Anthropic).
